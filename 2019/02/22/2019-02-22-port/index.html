80 端口 IIS 占用问题

闲来无事，学习nginx，可是下载下来启动nginx.exe发现，没有欢迎页面。

显示出来的却是微软iis页面，而后进行问题排查。

排查问题

  • cmd 进入命令提示符界面，netstat -ano， -ano查看当前系统所有网络连接状况，并且显示地址和端口
  • tasklist 则是命令查看当前进程。
  • 若是80端口的pid为4，名为System的进程占用，则查看是否是IIS占用了80端口

解决方法

  • 卸载IIS，步骤如下
    • 我的电脑 –> 添加或删除组件 –> 添加或删除[Windows]组件 –> IIS信息服务
      去掉 IIS前面的勾然后“下一步”就可以了。
  • 改iis的端口
    • 我的电脑 –> 右击-管理 –> iis管理 –> Default Web Site右击 –> 编辑绑定
  • 停止iis服务
    • 在管理员命令行运行iisreset/stop
    • CMD –> services.msc –> 服务里关闭World Wide Web Publishing Service
  • netstat -aon|findstr 80 而后 tskill 端口 杀死进程。
