1.1、 Lambda表达式

Lambda可以将函数作为一个方法的参数 (传递进方法中…)

语法格式:

​ (parameter) -> expression 或者 (parameter) -> {statements;}

PS :

​ 1) 如参数只有一个 可以不加圆括号;

​ 2) 不需要声明参数类型;

​ 3) 如果只有一条语法 可以不加花括号;

​ 4) 如果只有一条语句 编译器会自动将值返回;如果是多条的话 需手动 return;

1.2、 方法引用

方法引用为通过方法的名字来进行使用 或者 指向一个方法

语法格式:

​ 方法引用使用一对冒号 : :

​ 构造方法引用: 类::new

​ 静态方法引用: 类::静态方法

​ 实例方法引用: 类::实例方法 或者 对象::实例方法

1.3、 Stream API

Lambda表达式为 stream api 的基础. 可以多学学Lambda表达式

简单来说就是用流将元素一个一个进行处理 隐式进行

Stream API可以极大提高Java程序员的生产力，让程序员写出高效率、干净、简洁的代码。

这种风格将要处理的元素集合看作一种流， 流在管道中传输， 并且可以在管道的节点上进行处理， 比如筛选， 排序，聚合等。

元素流在管道中经过中间操作（intermediate operation）的处理，最后由最终操作(terminal operation)得到前面处理的结果。
